# daynote

个人日记/待办管理项目，使用 Timeline 系统管理日常事务。

## Skills

| Skill | 说明 | 触发词 |
|-------|------|--------|
| setup-timeline | 部署 timeline 的 Hermes 运行环境（首次使用前必须先完成） | 「部署 timeline」「setup timeline」 |
| timeline | 管理待办事项和事件，按日期组织 | 「提醒」「待办」「做完了」「今天有什么事」 |

## 使用流程

1. **首次使用**：加载 `setup-timeline` skill，完成 Hermes 环境部署
2. **日常使用**：直接使用 `timeline` skill 管理待办和事件
